== Description ==

This cursor plugin allows you to customize the cursor used on your blog, to make it more personal!

It's very simple to install - Simply upload the plugin, install it and replace skin.gif with your own image to customize the cursor!

== Installation ==

1. Upload `/jr-cursor/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Replace skin.gif with your own skin.gif image file.
4. Done!
